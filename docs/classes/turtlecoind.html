blocksDetailed

  • blocksDetailed(timestamp?: number, blockHashes?: string[], blockCount?: number): Promise<IBlocksDetailedResponse>
    • Defined in TurtleCoind.ts:80

    Retrieves up to 100 blocks. If block hashes are given, it will return beginning from the height of the first hash it finds, plus one. However, if timestamp is given, and this value is higher than any found in the array of blockHashes, it will start returning blocks from that height instead. The blockHashes array should be given the highest block height hashes first, then in descending order by height. First 10 block hashes go sequential, next in pow(2,n) offset (ie. 2, 4, 8, 16, 32, 64...), and the last block hash is always the genesis block. Typical usage: specify a start timestamp initially, and from then on, also provide the returned block hashes as mentioned above.

    Parameters

    • Default value timestamp: number = 0

      the timestamp to start from

    • Default value blockHashes: string[] = []

      the array of block hashes

    • Default value blockCount: number = 100

      the number of blocks to return

    Returns Promise<IBlocksDetailedResponse>

rawBlocks

  • rawBlocks(startHeight?: number, startTimestamp?: number, blockHashCheckpoints?: string[], skipCoinbaseTransactions?: boolean, blockCount?: number): Promise<IRawBlocksResponse>
    • Defined in TurtleCoind.ts:295

    Retrieves the raw hex representation of each block and the transactions in the blocks versus returning JSON or other encoded versions of the same.

    Retrieves up to 100 blocks. If block hash checkpoints are given, it will return beginning from the height of the first hash it finds, plus one. However, if startHeight or startTimestamp is given, and this value is higher than the block hash checkpoints, it will start returning from that height instead. The block hash checkpoints should be given with the highest block height hashes first. Typical usage: specify a start height/timestamp initially, and from then on, also provide the returned block hashes.

    Parameters

    • Default value startHeight: number = 0

      the height to start returning blocks from

    • Default value startTimestamp: number = 0

      the timestamp to start returning blocks from

    • Default value blockHashCheckpoints: string[] = []

      the block hash checkpoints

    • Default value skipCoinbaseTransactions: boolean = false

      whether to skip returning blocks with only coinbase transactions

    • Default value blockCount: number = 100

      the number of blocks to retrieve

    Returns Promise<IRawBlocksResponse>
